# PTDBProxyCore
This tool is for generating PT images for Doomtown
 ## Requirements
 Install .Net 5.0 Framework

 ## Usage
 From a command prompt or Powershell window, navigate into the folder with the tool and run the .exe followed by the starting card ID and the ending card ID that you want to generate images for

 e.g. "ptdbproxy-core.exe 25001 25056" would generate playtest images for the Debt of Blood expansion.
